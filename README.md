# Superstore Data Analyst Task

This repository contains my complete solution for the **Data Analyst Skill Evaluation Task** using the **Sample Superstore** dataset.[file:1]

## Tasks Covered

1. **Task 1 – Data Profiling**
   - Rows, columns, data types
   - Missing values, duplicates
   - Outliers in Sales and Profit
   - Relationships: Discount vs Profit, Sales vs Quantity
   - **Deliverable:** `profiling_report.pdf`[file:1]

2. **Task 2 – Data Cleaning**
   - Removed duplicates
   - Standardized date formats
   - Validated numeric fields
   - Fixed category/segment text issues
   - Added derived columns (Year, Month, Shipping Days, Profit Margin)
   - **Deliverables:** `cleaned_superstore.csv`, `data_dictionary.xlsx`[file:1]

3. **Task 3 – SQL Analytics**
   - 12+ queries for:
     - Total sales, profit, quantity
     - Monthly and YoY trends
     - Top products and customers
     - Category and region performance
     - Discount impact, loss-making items
     - Segment contribution, shipping time, outliers
   - **Deliverable:** `sql_queries.sql`[file:1]

4. **Task 4 – Analytical Insights**
   - At least 5 insights on:
     - Trends
     - Profitability
     - Customer/segment behaviour
     - Category performance
     - Risks and opportunities
   - Each insight includes: *What, Why, Action*
   - **Deliverable:** `insights_report.pdf`[file:1]

5. **Task 5 – Dashboard (Power BI)**
   - KPI cards: Total Sales, Total Profit, Avg Order Value, Best Region
   - Charts: Monthly Sales Trend, Profit by Category, Top 10 Products, Region Comparison, Discount vs Profit (scatter)
   - Filters: Region, Category, Segment, Order Date
   - **Deliverable:** `superstore_dashboard.pbix`[file:1]

6. **Task 6 – Executive Summary**
   - 1–2 page business summary covering:
     - Key performance highlights
     - High-profit categories and segments
     - Weak regions
     - Discount effect on profit
     - Risks and recommended actions
   - **Deliverable:** `executive_summary.pdf`[file:1]
